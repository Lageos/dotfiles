# Cheatsheet

## Unpack file
* unp ...

## Install / Remove Programms
* Add Repository: sudo add-apt-repository ppa:....
* Remove Repository sudo ppa-purge ppa:...

## Install program:
* Entpacken unp ...
* cd
* cat README.md | less
* ./configure oder ./autogen.sh
* make
* sudo checkinstall
